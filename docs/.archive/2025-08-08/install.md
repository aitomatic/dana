### Install Dana

Prerequisites:
- Python 3.9+

Install from PyPI:
```bash
pip install --upgrade dana-lang
```

Verify installation:
```bash
dana --version
```

Having trouble? Ensure your Python environment is active and that your PATH includes the Python user bin directory.
